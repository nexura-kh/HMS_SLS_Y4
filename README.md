# 🏨 Hotel Management System (HMS)

A professional **Hotel Management System** built using **C# Windows Forms** and **.NET Framework**, designed to simplify daily hotel operations such as room booking, customer management, staff records, and billing.  

This project demonstrates the use of **modular UI design**, **database connectivity**, and **object-oriented programming** principles in a Windows desktop environment.

---

## 🚀 Features

- 🧾 **Room Management** — Add, edit, or remove room types, check availability, and manage status (booked/vacant).
- 👥 **Customer Management** — Record customer details, booking history, and check-in/check-out logs.
- 👔 **Employee Management** — Manage staff records, roles, and work schedules.
- 💳 **Billing & Payment** — Automatically generate bills and handle payment processing.
- 📊 **Dashboard & Reports** — Visual insights into hotel performance and occupancy rate.
- 🔐 **User Authentication** — Secure login system for admin and staff.
- ⚙️ **Modular UI Design** — Easy-to-navigate dashboard with dynamic user controls.
